# Insurance_Premium_Predict
=======


## How to run:

1. Create an environment

```bash
conda create -n insurance python=3.8 -y


conda activate insurance

```

2. Activate the environment

```bash
conda activate insurance
```


2. install requirements

```bash
pip install -r requirements.txt
```
